• Βελτιώσαμε τον τρόπο με τον οποίο οι πληροφορίες σχετικά με τις εισερχόμενες, τις εξερχόμενες και τις αναπάντητες κλήσεις εμφανίζονται και συγχρονίζονται σε συνδεδεμένες συσκευές. Τώρα μπορείς να κάνεις τη σωστή επιλογή για όλες τις κλήσεις που δεν απάντησες εγκαίρως.
• Η προβολή "Όλα τα μέσα" έχει βελτιστοποιηθεί για να σου επιτρέπει να περιηγηθείς στο παρελθόν πολύ πιο γρήγορα στο μέλλον.
• Οι βελτιώσεις απόδοσης για ηχητικές κλήσεις σε περιβάλλοντα χαμηλού εύρους ζώνης θα σε βοηθήσουν να παραμείνεις σε σύνδεση σε μια εξαιρετική συνομιλία ακόμα και όταν η σύνδεσή σου δεν είναι καλή.
